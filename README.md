# CSGOHelper

https://csgohelper.pro

CSGOHelper is a website dedicated to Counter-Strike 2 (CS2) players, providing tools and resources to improve gameplay through grenade lineups, tactics, guides, and statistics.

### Home Page
- **URL**: https://csgohelper.pro/en-US/
- **Title**: CSGOHelper - CS2 nades, lineups, tactics in game Counter-Strike 2
- **Description**: The main landing page introducing the site. It highlights key features such as instant nade search by map and position, adding personal nades from YouTube, and building complete round tactics. Includes sections on "What you get", "How it works", and "What's new".
